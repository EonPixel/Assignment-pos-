Installation
Prerequisites:
PHP 8.0 or higher
Composer
MySQL database

Steps to Install:
1.cd pos
2.Install dependencies
   composer install
3.Generate application key
        php artisan key:generate
4.Run database migrations
        php artisan migrate
5.Run app
        php artisan serve
